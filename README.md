# REST API Documentation Sample

This repository is a technical writing portfolio sample for a fictional product called **SensorHub Cloud API**.

The sample shows how I document a REST API using:

- Markdown
- OpenAPI YAML
- JSON request and response examples
- Postman collection files
- Clear task-based documentation

## Product summary

SensorHub Cloud API lets developers connect gateways, read sensor data, manage alert rules, and troubleshoot device status.
